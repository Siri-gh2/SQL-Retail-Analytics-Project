# 📊 SQL Retail Analytics Project

A comprehensive SQL project covering end-to-end data analysis on a retail dataset. This project simulates real-world business scenarios, from raw data handling to advanced analytical insights.

---

## 🔹 Project Overview

This project includes:

* Database design (customers, stores, products, sales)
* Data generation (1000+ records)
* Data cleaning and transformation
* Analytical queries for business insights

---

## 🔹 Key Concepts Covered

### 🧹 Data Cleaning

* NULL handling using `IFNULL`, `COALESCE`
* Derived metrics like `net_revenue`
* Created a cleaned dataset (`sales_cleaned` view)

### 🔍 Data Analysis

* Filtering using `WHERE`
* Aggregation using `GROUP BY`
* Post-aggregation filtering using `HAVING`

### 🧠 Advanced SQL

* `CASE WHEN` for segmentation
* Joins across multiple tables
* Window functions (`RANK`, `LAG`, `LEAD`, `NTILE`)
* FIRST_VALUE / LAST_VALUE analysis

### 📅 Time-Based Analysis

* Daily & monthly trends
* Growth calculations
* Customer activity tracking

---

## 🔹 Sample Insights

* Revenue distribution across stores and categories
* High-value customers and transactions
* Customer segmentation (Premium, Gold, etc.)
* Sales trends and growth patterns
